Tienda de Suplementos:
Este es un proyecto simple de comercio electrónico desarrollado utilizando HTML, CSS y JavaScript. Simula la funcionalidad básica de una tienda de suplementos con un carrito de compras interactivo y persistente.
Funcionalidades:
* **Visualización de Productos:** Muestra un listado de suplementos con nombre, categoría, descripción y precio.
* **Carrito de Compras:** Permite agregar productos al carrito.
* **Persistencia de Datos:** Utiliza **Local Storage** para recordar los productos en el carrito incluso después de recargar la página.
* **Gestión del Carrito:**
    * Cálculo y visualización del total de la compra.
    * Botón para **Vaciar el Carrito** por completo.
    * Botones para **Eliminar** productos individualmente del carrito.
    * Botón para **Finalizar Compra** (con un mensaje de confirmación simple).

Instalación y Uso:

No se requiere instalación de librerías o dependencias externas.

1.  **Clonar o Descargar:** Descarga o clona este repositorio en tu máquina local.
2.  **Abrir:** Abre el archivo `index.html` directamente en tu navegador web.
3.  **Interactuar:** Empieza a hacer clic en los botones "Comprar" para agregar productos y prueba las funcionalidades del carrito.
